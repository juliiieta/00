1- ¿Cuáles son los tipos de títulos que hay?

Los tipos de titulos que hay son H1, H2, H3, H4, H5 y H6.

2- ¿Cuál es la importancia del número que lleva la etiqueta de título?

La importancia esta ordenada desde el mas importante H1 hasta el menos H6.

Se debe evitar colocar los titulos en desorden, es decir escribirlos en orden de importatancia asi el dispositivo de lectura los lee correctamente.

5- ¿Qué es un atributo en una etiqueta HTML? Dar dos ejemplos de etiquetas HTML con atributos, marcando sus respectivos atributos.

Un atributo es una caracteristica. Un atributo de la etiqueta "html" puede ser "lang" y su valor "es".O un atributo de la etiqueta "meta" puede ser "name" con el valor "viewport".

6- ¿Qué atributos tienen las etiquetas de título?. Definir.

Las etiquetas de titulo usan atributos globales, lo que significa que se pueden usar en varias etiquetas, tales como id y class.

ID: Tiene el proposito de identificar el elemento.

CLASS: Es una lista de las clases de un elemento.

7- ¿Qué hace la etiqueta "P"? ¿Cuáles son sus atributos?

La etiqueta p permite escribir un parrafo. Sus atributos son title, style, id, class, dir y lang.

2- ¿Qué es el HTML? ¿En qué consiste?

HTML es un lenguaje de marcado de hipertexto. Define la estructura de una WEB. "Hipertexto" hace referencia a los enlaces que conectan páginas web entre sí. Consiste en una serie de elementos y etiquetas que organizan y pueden modicar el contenido.

3- ¿Qué es una etiqueta? ¿Funciona si se escribe tanto en mayúscula como en minúscula? ¿Cuál es la mejor práctica para escribir etiquetas?

Una etiqueta se usa para crear un elemento. Funcioina en ambos casos (Mayuscula y miniscula) pero la manera correcta de hacerlo para que el codigo sea universal es en minuscula.

4- ¿A qué llamamos elemento HTML? ¿Cuál es su anatomía? Explicar sus partes.

Es un elemento de una WEB con su respectivo contenido. Su anotomia esta fromada por una etiqueta de apertura, su cintenido y una etiqueta de cierre. La etieta de apertura funciona para indicar de que forma mostrar el conntenido. Y la de cierre para terminar esta iondicacion.
